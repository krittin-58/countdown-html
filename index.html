? How to Use

Countdown Timer — a free, full-screen timer that runs entirely in your browser. No account or installation needed.

➕ Adding a Timer

  1. Click ＋ ADD to open the timer form.
  2. Optionally type a name and enter the duration in minutes.
  3. Click ▶ START — or pick a quick preset (5m, 10m, 15m, 25m 🍅, 45m).

⏱ Timer Controls

⏸ PAUSEFreeze the timer
▶ RESUMEContinue a paused timer
↺ RESETReturn to original duration
✕Remove the timer card
+1m / +5mAdd time to a running or paused timer
🔁Loop — restart automatically when done
⏱Stopwatch — switch to count-up mode
↑ / ↓Move timer up or down in the list
⧉Duplicate the timer card

🍅 Pomodoro & Sequential

  • 🍅 POMO — starts an automatic Work / Break cycle (25 min work · 5 min short break · 15 min long break).
  • ⚙️ next to POMO — customise Work, Short Break, Long Break durations and interval.
  • ↕ SEQ — when enabled, the next timer starts automatically when the current one finishes.

🔧 Other Features

  • ⊞ SETS — save and reload groups of timers.
  • 🔗 SHARE — copy a URL or embed code for the current timers.
  • 📊 STATS — view completed timers, 7-day focus chart, daily goal, and export CSV.
  • HIDE / click display — hide the control panel for a clean full-screen view.
  • Theme dropdown — Auto (follows OS), Dark, Light, Ocean, Sunset, Forest, or Purple.
  • 🔊 slider — adjust alarm volume; choose Beep, Bell, Chime, or Silent (preview plays on change).
  • Aa slider — scale the timer digits larger or smaller.

⌨️ Keyboard Shortcuts

SpacePause / Resume first active timer
NOpen / close the Add Timer form
RReset the first timer
APause all running timers
SResume all paused timers
FToggle fullscreen
EscClose an open modal; otherwise hide / show controls

Your timers and settings are saved automatically in your browser's local storage.

🍅 โพโมโดโร & โหมดลำดับ
